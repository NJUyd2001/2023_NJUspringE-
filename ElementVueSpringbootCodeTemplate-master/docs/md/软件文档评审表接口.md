## 搜索

- 根据**SDRID**搜索

网段：`/softwareDocReview/select/bySDRID`

参数：

``` js
{
    "SDRID":""
}
```

- 根据**PID**搜索

网段：`/softwareDocReview/select/byPID`

参数：

``` js
{
    "PID":""
}
```
