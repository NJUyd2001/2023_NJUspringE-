## 搜索

网段：`/softwareDocReview/select/bySDRID`

参数：

``` js
{
    "SDRID":""
}
```

{
    "PID":""
}

## 更新

- 更新主体信息

网段：`/softwareDocReview/update/main`

参数：

``` js
{
            SDRID:'', // 必须提供SDRID
            SoftWareName:'',
            VersionNumber:'',
            Client:'',
            ReviewCompleteDate:'',
            Reviewer:'',
            Examiner:''
}
```

返回字符串提示

- 更新**tableData**信息

网段：`/softwareDocReview/update/item`

参数：

``` js
{
            ID:'', // 必须提供ID，可由select返回信息获得
            ReviewCategories: '',
            Reviewitem: '',
            ReviewContent: '',
            ReviewResultExplanation: '',
            radio: ''
}
```

返回字符串提示

## 删除

网段：`/softwareDocReview/delete`

参数：
``` js
{
    SDRID:''
}
```

返回字符串提示
